# AI-Based Skill Assessment and Course Recommendation System

## Overview
This is a full-stack web application that assesses users' technical skills through quizzes and recommends personalized courses based on their performance.

## Features
- User Login
- Skill Assessment Quiz
- AI-Based Course Recommendations
- Admin Dashboard
- PDF Report Generation

## Tech Stack
- React
- TypeScript
- Vite
- Python
- Flask
- Google Gemini API

## Installation

1. Clone the repository
2. Run:
   ```bash
   npm install
   ```
3. Add your Gemini API key in `.env.local`
4. Start the application:
   ```bash
   npm run dev
   ```
